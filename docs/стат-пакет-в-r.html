5 Стат пакет в R

R - это язык программирования для статистических вычислений и графического представления информации, бесплатное программное обес Пользователи Windows и macOS могут скачать R на сайте CRAN (Comprehensive R Archive Network). Пользователи Linux и Unix могут установить пакеты R с помощью своего инструмента управления пакетами.

5.1 Windows

  1. Перейдите на страницу http://www.r-project.org/ в своем браузере.

  2. Нажмите CRAN. Вы увидите список зеркал сайта, упорядоченных по странам.

  3. Выберите ближайший к вам сайт или самый первый указанный в верхней части списка (0-Cloud), который, как правило, хорошо работает для большинства ченных по странам.

  4. Выберите ближайший к вам сайт или самый первый указанный в верхней части списка (0-Cloud), который, как правило, хорошо работает для большинства местоположений.

  5. Нажмите Download R for (Mac) OS X (Скачать R для (Mac) OS X).

  6. Нажмите на файл .pkg для последней версии R в разделе Latest release (Последний выпуск), чтобы скачать его.

  7. По завершении загрузки дважды щелкните файл .pkg и ответьте на привычные вопросы.

Для установки RStudio необходимо загрузить последнюю версию установщика для вашей платформы с сайта RStudio: (https://rstudio.com/products/rstudio/download/). Версию RStudio Desktop с открытым исходным кодом можно скачать и использовать бесплатно.

Примечание:

Просмотр документации: help.start()

Быстрый вызов функции: ?mean

Поиск текстового шаблона функции: ??mean
