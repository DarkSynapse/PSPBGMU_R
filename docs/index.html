            В данном учебном пособие рассмотрены типы данных в языке программирования R, этапы проведения мета-анализа как части систематического обзора, методические рекомендации по проведению мета-анализа на языке программирования R.

            Учебное пособие предназначено для студентов лечебного, стоматологического, педиатрического факультетов и факультета клинической психологии, для студентов цифровой кафедры. Пособие может быть полезно клиническим ординаторам и аспирантам при статистической обработке большого объема информации, представления обобщенных клинических данных и их статистического анализа.
